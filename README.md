## Download & Run Locally

Clone the project

```bash
 git clone https://github.com/nikhilranga4/BeyondChats-Chatbot
```

Go to the project directory

```bash
  cd BeyondChats-Chatbot-main
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
